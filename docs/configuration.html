Configuration

Currently, the Solosis CLI can process several types of data files.

Data files

  • RNA-sequnce R1 & R2 (.fastq)

  • Multiome GEX & ATAC (.fastq)

  • Cellranger outputs (.h5??)

  • Input samplefile csv or txt files

I am leaving this in for now, so that I canrefer back to this at a later date.

Project

projects:
  - project: project_1
    datasets:
      ...
    args:
      h5ad:
        batch_processing: True

Multiple projects can be defined in a single parameters file, and each can define multiple datasets. Each project item is defined by the following keys:

sample_id

sample_name

irods_path
