cli

Command line utility for the Cellular Genetics programme at the Wellcome Sanger Institute

cli [OPTIONS] COMMAND [ARGS]...

Options

--version

Show the version and exit.

alignment

Commands for running alignment tools.

cli alignment [OPTIONS] COMMAND [ARGS]...

cellranger

Run Cell Ranger for single-cell RNA sequencing alignment and analysis

Cell Ranger (7.2.0) performs sample demultiplexing, barcode processing, and gene counting for single-cell 3’ and 5’ RNA-seq data, as well as V(D)J transcript sequence assembly

cli alignment cellranger [OPTIONS]

Options

--sample <sample>

Sample ID (string)

--samplefile <samplefile>

Path to a CSV or TSV file containing sample IDs

--create-bam

Generate BAM files for each sample

--version <version>

Cell Ranger version to use (e.g., ‘7.2.0’)

cellrangerARC

CellrangerARC aligns GEX & ATAC seq reads…

Cell RangerARC (2.0.2) Software suite designed for analysing & interpreting scRNA seq data, including multi-omics data.

cli alignment cellrangerARC [OPTIONS]

Options

--samplefile <samplefile>

Required Sample file text file

--libraries <libraries>

Required libraries file CSV
